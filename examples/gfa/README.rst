Group Factor Analysis (GFA)
---------------------------

These examples demonstrate Group Factor Analysis, a Bayesian method for
discovering shared and group-specific latent structure in multi-group data.

GFA uses Automatic Relevance Determination (ARD) to automatically select
the appropriate latent dimensionality during inference.

Start with :ref:`sphx_glr_auto_examples_gfa_plot_1_simulation.py` to understand
the generative model, then proceed through fitting and posterior analysis.
